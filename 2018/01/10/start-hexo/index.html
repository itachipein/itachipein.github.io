第一次搭建博客，为防自己忘记，记录一下流程：

一、安装Git
二、创建github仓库
三、安装node.js
四、安装hexo
五、hexo初始化与配置
六、本地启动
七、部署到github
八、主题下载
九、增加日志
十、域名修改

一、安装Git
略

二、创建github仓库
这是部署到github比较关键的一步，注册github，并在个人主页下创建仓库，注意仓库命名为：githubname.github.io

三、安装node
略

四、安装hexo
1、安装完node之后可以直接用npm命令安装。在任意位置打开命令行工具（或者git bash）
2、npm install -g hexo

五、hexo初始化与配置
1、在任意位置创建文件夹，如我的D:\study\hexo_blog；
2、在上面创建的文件夹（hexo_blog）下打开命令行工具（或者git bash），输入命令 hexo init；
3、配置_config.yml，其中比较重要的是theme和deploy，前者决定主题，后者则是部署到github
deploy:
type: git
repo: https://github.com/itachipein/itachipein.github.io.git
branch: master

六、本地启动
hexo g
hexo s
然后访问localhost:4000

七、部署到github
1、npm install hexo-deployer-git –save
2、hexo clean
3、hexo g
4、hexo d
部署完成后访问https://githubname.github.io即可访问
更多hexo命令参考：https://segmentfault.com/a/1190000002632530

八、主题下载
以我使用的next主题为例：
git clone https://github.com/theme-next/hexo-theme-next.git themes/next
更多主题
clone下来后在_config.yml中将theme改成相应主题即可
同时主题也有自己的_config.yml，next主题更多配置参考http://theme-next.iissnan.com/getting-started.html

九、增加日志
在source/_posts下执行：
hexo new yourFileName
默认是md文件，在里面修改即可

十、域名修改
如果不做这一步，使用的是默认域名githubname.github.io，如果想改成自己想要的域名：
1、申请域名（阿里云等渠道）
2、登录进入万网的域名控制台，点击”域名和网站”中的”云DNS”
3、点击对应域名的”解析”
4、点击添加解析，记录类型选A和CNAME都要
（1）A记录的是ip地址，github(官方文档)提供了两个IP地址，192.30.252.153和192.30.252.154，这两个IP地址为github的服务器地址，两个都要填上，解析记录填@，线路默认
（2）CNAME记录值填github博客网址，解析记录填www，线路默认。
5、全部设置完成后在hexo根目录的source文件夹里创建CNAME文件，不带任何后缀，里面添加域名信息，如：ljm_zjuer.top。实践证明如果填写的是www.ljm_zjuer.top那么以后你只能用www.ljm_zjuer.top访问，而如果填的是ljm_zjuer.top。那么用www.ljm_zjuer.top和ljm_zjuer.top访问都是可以的。重新清理hexo,并发布即可用新的域名访问。
更多参考：https://www.cnblogs.com/penglei-it/p/hexo_domain_name.html
